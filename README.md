# Dictionary
A simple offline-first dictionary app. Kotlin stack:
> Jetpack Compose,
> Clean architecture,
> MVI,
> DI with Koin,
> Room for offline caching,
> Retrofit for API calls

Free dictionary API: https://dictionaryapi.dev/
